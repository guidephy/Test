📰 素養閱讀：從 SpaceX 星艦第11次試飛看牛頓三大運動定律

2025年10月，SpaceX 的超重型火箭「星艦」（Starship）圓滿完成第11次試飛：推力高達 74,000 kN 的33具「猛禽」（Raptor）引擎同時點火，將質量超過 5,000 公噸 的整組飛行器從德州博卡奇卡基地推入太空。這場壯觀的升空背後，正是牛頓在1687年提出的三大運動定律在現代工程中的完美體現。

⚖️ 牛頓第一定律（慣性定律）

「物體在不受外力（或合力為零）的情況下，靜止的物體保持靜止，運動的物體保持等速直線運動。」星艦進入太空軌道後，若關閉引擎，幾乎無空氣阻力，將依慣性持續飛行——這正是衛星能長期繞地球運行的基本原因。

⚡ 牛頓第二定律（運動定律）

「物體所受合力等於質量乘以加速度：$\vec{F}_{net} = m\vec{a}$。」星艦起飛時，引擎提供的推力 $F$ 須大於重力 $mg$ 才能加速上升。合力 $F_{net} = F_{推} - mg$ 決定加速度大小。下圖左顯示不同質量下，加速度隨合力增大的線性關係。

🔄 牛頓第三定律（作用力與反作用力）

「當物體A對物體B施一力，B必對A施一大小相等、方向相反的力。」猛禽引擎將燃燒氣體高速向下噴出，氣體同時對火箭施一向上的反作用力——這就是推力的來源。兩力大小相等但作用在不同物體上，永遠不能相消。

從圖右可看出，三大定律在火箭科學中各有不可缺少的角色：第一定律解釋了為何太空中不需要引擎持續推進；第二定律計算了引擎推力如何轉化為加速度；第三定律則是推進力的根本來源。三者環環相扣，少了任何一個，星艦就無法升空。

以具體數字為例：假設星艦起飛質量約 $m = 5000 \ \text{t} = 5 \times 10^6 \ \text{kg}$，引擎總推力 $F = 7.4 \times 10^7 \ \text{N}$，重力加速度 $g = 9.8 \ \text{m/s}^2$，則合力為：

$$F_{net} = F_{推} - mg = 7.4\times10^7 - 5\times10^6 \times 9.8 \approx 2.4\times10^7 \ \text{N}$$

初始加速度約為 $a = \dfrac{F_{net}}{m} = \dfrac{2.4\times10^7}{5\times10^6} \approx 4.8 \ \text{m/s}^2$，大於零，表示火箭確實能向上加速。隨著燃料燃燒消耗，質量 $m$ 不斷減小，加速度也會隨之增大——這便是火箭飛行後段會急速加速的原因，稱為「齊奧爾科夫斯基效應」。

作用力與反作用力：大小相等、方向相反，且分別作用在兩個不同的物體上，因此永遠不能互相抵消。
Bloom：記憶層次——辨識第三定律的定義。
2.理解 Understand
根據文章，星艦進入太空並關閉引擎後，幾乎沒有空氣阻力，它最接近哪種運動狀態？這是哪一定律的體現？
正確答案：（B）等速直線運動；牛頓第一定律
關閉引擎後合力趨近零（無空氣阻力），依慣性（第一定律）保持等速直線運動。這也是衛星長期運行的基礎。
Bloom：理解層次——將定律對應到太空飛行情境。
3.理解 Understand
下列四種情境中，哪一對力屬於牛頓第三定律的「作用力與反作用力」？
正確答案：（B）
（B）符合第三定律：火箭對氣體向下施力，氣體對火箭向上施反作用力，兩力大小相等、方向相反、作用在不同物體（火箭與氣體）。
（A）是「平衡力」（作用在同一物體書上），不是作用力反作用力對。
Bloom：理解層次——辨別第三定律力對與一般平衡力的差異。
4.應用 Apply
一物體質量 $m = 5 \ \text{kg}$，受合力 $F_{net} = 30 \ \text{N}$，求其加速度為何？（$g = 10 \ \text{m/s}^2$）
公式：$a = \dfrac{F_{net}}{m}$
正確答案：（B）$6 \ \text{m/s}^2$
$a = \dfrac{F_{net}}{m} = \dfrac{30}{5} = 6 \ \text{m/s}^2$
Bloom：應用層次——直接代入第二定律。
5.應用 Apply
一輛質量 $1200 \ \text{kg}$ 的電動車，以引擎推力 $F = 4800 \ \text{N}$ 在水平路面行駛，路面摩擦力為 $f = 1200 \ \text{N}$。此車的加速度為多少？
合力 $F_{net} = F - f$；$a = \dfrac{F_{net}}{m}$
正確答案：（A）$3 \ \text{m/s}^2$
$F_{net} = 4800 - 1200 = 3600 \ \text{N}$
$a = \dfrac{3600}{1200} = 3 \ \text{m/s}^2$
Bloom：應用層次——考慮摩擦力計算合力再求加速度。
6.應用 Apply
某人站在體重計上，體重計顯示 60 kg（即正向力 $N = 600 \ \text{N}$）。若此人搭乘電梯向上加速（$a = 2 \ \text{m/s}^2$），體重計的讀數將會如何變化？（$g = 10 \ \text{m/s}^2$）
提示：$N - mg = ma$，故 $N = m(g+a)$
正確答案：（B）顯示 720 N（變重）
向上加速時，正向力 $N = m(g+a) = 60\times(10+2) = 60\times12 = 720 \ \text{N}$，體重計讀數「增大」，俗稱「超重」。
Bloom：應用層次——電梯加速情境的第二定律應用。

🧮 三、計算與分析題（每題 15 分，共 30 分）
9.分析 Analyze
（分析題）參考文章數據，星艦起飛時質量 $m = 5\times10^6 \ \text{kg}$，引擎推力 $F = 7.4\times10^7 \ \text{N}$，$g = 9.8 \ \text{m/s}^2$。

（a）計算星艦起飛瞬間所受的重力大小。
（b）計算合力 $F_{net}$ 及起飛加速度 $a$。
（c）燃料燃燒後，若質量減少為 $m' = 3\times10^6 \ \text{kg}$（推力不變），此時加速度為何？並說明為何火箭越飛越快。
詳解：
（a）重力 $W = mg = 5\times10^6 \times 9.8 = 4.9\times10^7 \ \text{N}$

（b）$F_{net} = F - W = 7.4\times10^7 - 4.9\times10^7 = 2.5\times10^7 \ \text{N}$
$a = \dfrac{F_{net}}{m} = \dfrac{2.5\times10^7}{5\times10^6} = 5 \ \text{m/s}^2$

（c）$W' = 3\times10^6 \times 9.8 = 2.94\times10^7 \ \text{N}$（但高空 $g$ 略小，簡化取 $\approx 9.8$）
$F_{net}' = 7.4\times10^7 - 2.94\times10^7 = 4.46\times10^7 \ \text{N}$
$a' = \dfrac{4.46\times10^7}{3\times10^6} \approx 14.9 \ \text{m/s}^2$

說明：由 $a = F_{net}/m$，分母 $m$ 隨燃料燃燒而減小，分子 $F_{net}$ 因重力減小而增大，加速度因此持續增大，火箭越飛越快。
Bloom：分析層次——分段計算並解釋火箭加速機制。

參考答案：
（a）反駁：牛頓第三定律的作用力與反作用力，是指火箭對氣體施力，氣體同時對火箭施反作用力。這對力作用在兩個不同的物體（火箭和噴出的氣體）之間，完全不需要借助外在介質。真空只是沒有空氣，但噴出的燃氣本身仍然存在，因此火箭噴氣在太空中同樣有效。

（b）生活中的第三定律例子（參考）：
① 游泳：手向後撥水（手對水向後施力），水對手向前施反作用力，推動身體前進。
② 氣球放氣：氣球對空氣向後噴出施力，空氣對氣球向前施反作用力，氣球飛出。
③ 步行：腳對地面向後踩（腳對地施力），地面對腳向前的摩擦力（反作用力）推動人前進。
